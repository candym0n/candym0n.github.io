# Artistic Canvas Website

A modern, artistic website showcasing interactive animations and creative design elements.

## Features

- Interactive particle canvas with mouse interactions
- Floating animated elements with parallax effects
- Color transitions and dynamic themes
- Responsive layout built with Bootstrap
- Modular JavaScript architecture
- Well-organized CSS with separate concerns

## Configuration

The website is highly configurable through the `js/config.js` file:

- Adjust canvas particle settings
- Modify animation speeds and behaviors
- Change theme colors and transitions
- Toggle features on or off

## Browser Compatibility

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Dependencies

- Bootstrap 5.3.0
- No other external dependencies
